Windows10系统添加回中文输入法中美式键盘

微软从Windows8开始就默认删除了中文输入法中的美式键盘。如需添加回来可以通过修改注册表实现。

该方法同样适用于WIndows8、WIndows8.1系统。

新建一个文本文档.txt文件，输入以下内容

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Keyboard Layout]

[HKEY_CURRENT_USER\Keyboard Layout\Preload]
"1"="00000804"

[HKEY_CURRENT_USER\Keyboard Layout\Substitutes]
"00000804"="00000409"

[HKEY_CURRENT_USER\Keyboard Layout\Toggle]

然后改文本文档.txt的后最txt为reg。双击文本文档.reg把注册表添加到系统中，然后注销下用户再登陆就能像WindowsXP、Windows7那样按Ctrl+Shift切换输入法了。

附：文本文档可以是其他名字只要后缀最后要改成reg即可
