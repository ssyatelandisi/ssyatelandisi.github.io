AdGuard，远离广告、弹窗和挖矿！

AdGuard简述

现在网上看个视频一般都会遇到视频正片开始前植入的广告，短则几秒，长则数分钟。还有有些铺天盖地的广告，很是令人忧心。更令人恶心的是现在有些网站为了提升收益，植入挖矿脚本，当你访问的时候就会利用你的电脑进行挖矿，有时会造成电脑严重卡顿。

而AdGuard可以让你远离上述烦恼。它是摆脱烦人广告，在线跟踪并使您的计算机原理恶意软件的最佳途径。其使您体验快速，安全，无广告的网络冲浪。可以说当前最强大好用的去广告软件了，不仅能去掉页面广告，还能阻止广告等弹窗，也可以阻挡挖矿行为。

AdGuard拥有Windows、Mac、Android、iOS以及浏览器拓展等诸多版本。本篇主要以Windows版为例予以介绍。

AdGuard for Windows下载及安装

AdGuard for Windows 不只是广告拦截程序，它还组合了可获取最佳网络体验所需功能的多用途工具。其可拦截广告和危险网站，加速网页载入，保护儿童的在线安全。——AdGuard官网描述

视频广告拦截

通常情况下，视频网站的广告也都能被拦截。由于视频网站规则经常更新，有时拦截会造成视频正片不加载或其他效果。这时你可以点击AdGuard助手，先暂停过滤30秒。再刷新下网页使其不处于广告拦截状态。当然，可以报告此网站，按提示信息填写一份报告提交给开发人员，让他们尽早了解和做出该网站的拦截方法。

AdGuard进阶功能

点击主界面的设置可以看到更多功能

隐形模式

隐形模式主要是用来防止跟踪和信息泄露。

开启后才可以设置需要隐匿项。本片着重讲隐藏您的 User-Agent和隐藏您的 IP 地址

隐藏您的 User-Agent

主用用于虚拟一个UA标识

很多网站的回复都或显示UA标识

当往隐藏您的 User-Agent中填入

Mozilla/5.0 (iPhone; CPU iPhone OS 11_4_1 like Mac OS X) AppleWebKit/605.1.15 (KHTML, like Gecko) Version/11.0 Mobile/15E148 Safari/604.1

再用浏览器去访问百度时会发现，明明是电脑端的浏览器，打开百度却是移动端的样子去了

利UA分析在线工具查看会发现当前明明是WIndows10系统谷歌浏览器却显示成了iphone设备Safari浏览器了

隐藏您的 IP 地址

设置一个IP地址后，对于获取本地IP地址的Javascript脚本获取都设置的IP，从而达到隐藏IP地址的目的。但不同于网络代理可以隐匿外网访问时的IP。

要注意点，当IP设置成了国外的或某些特殊IP后，会导致网易云音乐以及一些视频网站的视频不能播放，会提示你说在国家或地区因版权方要求，无法播放。

拓展

在谷歌浏览器和火狐浏览器等支持拓展插件的浏览器上可以安装暴力猴或油猴拓展插件，然后添加脚本让浏览器功能更强大。但一般都无法顾及IE浏览器。如果用AdGuard的拓展功能添加脚本也可达到解决此遗憾。

以Greasy Fork上的一款京东晒单查看大图的脚本为例

点击添加拓展程序

粘贴Greasy Fork上的京东晒单查看大图脚本的的链接地址，再点击安装

等加载完后再点击安装

安装成功后点击关闭即可

此时包括IE在内的所有浏览器都能使用该脚本了
